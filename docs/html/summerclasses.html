Python:

Thursdays 11:30-12:30 (via Zoom)

This fun summer class introduces students to Python, one of the most popular and powerful programming languages today. We will cover the fundamental concepts in programming in Python while creating projects easy to customize. This class is held via Zoom and students will have access to class recordings for review or follow-up if a vacation coincides with a class day.

  • Required book: Coding Projects in Python, DK Publishing
  • Required tech: an ability to set a monitor to split-screen view so the student can see both the teacher and their coding screen simultaneously.

Course Outline:

Unit 1 Introducing Python and understanding variables, conditionals, loops, lists, and functions.
Unit 2 Projects with Turtle Graphics: Robot Builder, Kaleido-Spiral, Starry Night, and Mutant Rainbows.
Unit 3 Playful Apps: Reading and Writing Files, Calendars, Cryptography, and Screen Pets.
Unit 4 Games in Python: Caterpillar (or Snake), Snap, Matchmaker, and Egg Catcher

Tuition:

$300 new students, $250 current or former students.
Tuition covers 11 sessions. There is one mandatory in-person meetup on June 7th to install Python and the editor we'll be using for class and to answer any questions from parents and students.

Intermediate Scratch 3.0

Wednesdays 10:00 - 11:00 (via Zoom)

Ready to take your Scratch programming skills to the next level? Join us for this weekly Zoom coding class and complete 8 in-depth game units from Scratch 3 Programming Playground, by Al Sweigart. Explore the elements of what makes a great game and how to define a game's objectives, rules, and goals. Develop maze games, puzzles, racing challenges and platformer games, and develop a portfolio of final versions to share on-line with friends and family.

  • Required book: Scratch 3 Programming Playground, Al Sweigart
  • Required tech: an ability to set a monitor to split-screen view so the student can see both the teacher and their coding screen simultaneously.

Course Outline:

Unit 1 Student profile setup, class studio introduction, Scratch review, standard class rules and expectations.
Unit 2 Underwater Treasure Hunt
Unit 3 Mazes
Unit 4 Cloning
Unit 5 Gravity in Games
Unit 6 Game Themes and Animating Sprites
Unit 7 Platformer Games
Unit 8 Racing Games

Tuition: $300 new students, $250 current or former students
